**Asosiy tushunchalar:**

1. **Authentication**: Foydalanuvchining kimligini tasdiqlash (login)
2. **Authorization**: Foydalanuvchi nimalarga ruxsatli ekanligini belgilash
3. **RBAC**: Rollarga ruxsat beriladi, foydalanuvchilar rollarga biriktiriladi
4. **Least Privilege**: Foydalanuvchiga faqat kerakli ruxsat berish

**Savollarga javob:**

1.Why do we need roles instead of assigning permissions to individual users?

a:Rollardan foydalanish, har bir foydalanuvchiga alohida ruxsat berishdan qulayroq, boshqaruv osonlashadi.

2.What's the difference between authentication and authorization?

a:Authentication = kimligini tekshirish, Authorization = nima qilishi mumkinligini belgilash.

3.Give 3 real-world examples of role-based access control

a:Misol: bank xodimlari (ATM, depozit, audit), kompaniya xodimlari (HR, Admin, Sales), forum/moderatorlar.
